# Restaurant Order Tracking System (POS)

A web-based **restaurant order and cashier management system** developed using **PHP** and **MySQL**.

This project is designed to help small and medium-sized food businesses manage tables, orders, payments, and daily revenue in a simple and efficient way.

---

## Features

- User authentication and role-based authorization
- Table-based order management
- Real-time order tracking
- Cashier payment and checkout system
- Daily revenue (turnover) reporting
- Simple and fast POS-style interface

---

## Roles

- **Admin**
  - Full system access
  - User and table management
  - Revenue and report monitoring

- **Cashier**
  - Order payment and checkout operations
  - Table-based billing

- **Waiter**
  - Order entry and table management (limited access)
